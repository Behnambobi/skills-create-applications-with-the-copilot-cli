# سیاست امنیتی

## نسخه های پشتیبانی شده

از این بخش برای اطلاع دادن به مردم در مورد نسخه های پروژه شما استفاده کنید
در حال حاضر با به روز رسانی های امنیتی پشتیبانی می شود.

| نسخه | پشتیبانی می شود          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## گزارش یک آسیب پذیری

از این بخش استفاده کنید تا به مردم بگویید چگونه یک آسیب پذیری را گزارش کنند.

به آنها بگویید کجا بروند، هر چند وقت یکبار می توانند انتظار دریافت به روز رسانی در a را داشته باشند
آسیب پذیری گزارش شده، در صورت پذیرفته شدن آسیب پذیری چه انتظاری باید داشت یا
رد شد و غیره
